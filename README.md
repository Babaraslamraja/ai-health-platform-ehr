#   AI Health Platform – EHR Integration Project

##  Overview
This project simulates a platform-based Electronic Health Record (EHR) system by integrating data from multiple healthcare sources into a unified patient-level dataset.

It demonstrates how fragmented healthcare data can be transformed into a shared care record, enabling interoperability and supporting future AI-driven healthcare solutions.

##  Objectives
- Integrate multiple EHR datasets
- Perform data cleaning and transformation
- Create a unified patient dataset
- Generate a shared care record
## Project Highlights

- Integrated multiple EHR systems into a unified dataset
- Applied data cleaning and transformation techniques
- Selected the most recent patient encounters
- Created a shared care record for cross-system visibility
- Simulated a real-world healthcare data pipeline
